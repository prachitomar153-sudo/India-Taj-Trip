# India Taj Trip Website

Professional travel and tourism website developed for showcasing tour packages, travel destinations, and tourism services across India.

## Features

* Responsive website design
* Tour package pages
* Destination showcase sections
* Travel inquiry forms
* Mobile-friendly layout
* SEO-optimized structure
* Fast loading pages

## Technologies Used

* PHP
* HTML5
* CSS3
* JavaScript

## Purpose

This project was created to establish a professional online presence for a travel and tourism business and help users explore and book travel experiences across India.
